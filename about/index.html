5. 使用 Jenkins + 蒲公英 实现持续集成，自动打包脚本
6. 制定 GitFlow 规范
我的职责：
1. 核心功能模块的研发如：考试系统、埋点方案、审批、业务统计与微站统计、登录优化等
2. 维护后台工单系统，帮助业务员和客户解决问题
3. 分析并修复 Sentry 记录的 Crash Log，使崩溃率显著降低
4. 参与项目评审，参与集成测试，负责版本发布
5. Code Review、工作汇报等

5. Objective-C 与 JavaScript 交互
我的职责：
